Question-1: What are the different ways to select an element in the DOM?

In JavaScript, we can select DOM elements in many ways like getElementById(id), getElementsByClassName(class), getElementsByTagName(tag), querySelector(selector), querySelectorAll(selector) etc. Here, getElementById(id) is used to select an element by its id name; getElementsByClassName(class) is used to select an element by its id class name; getElementsByTagName(tag) is used to select an element with its tag name; querySelector(selector) is used to selects the first matching element using a CSS selector. Among all the methods these are the most popular and commonly used.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML returns or sets the HTML content of a element with child elements and tags. innerText returns or sets the visible text of an element but don't show the hidden elements and styling. On the other hand textContent returns or sets the entire text content with all hidden elements.

Question-3: What is event delegation in the DOM?

In the DOM, event delegation is a process where the parent element handles events for its child elements. Though the events are added dynamically. Instead of attaching event listeners to multiple child elements we can add a single event listener to a common anchestor. It is commonly used for handling clicks on lists, tables etc.

Question-4: What is event bubbling in the DOM?

Event bubbling means a way where events travel in the DOM when something happens like click or etc. When we click a element, the event first take places on that element and then it "bubbles up" to its parent element, then its parent's parent and so on until it reaches the html tag. This is called event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript we can create a element by using document.createElement() method. And we can add a element by using appendChild() or append() method. And we can remove a element by using removeChild() or remove() method.
